nmap的使用

  • 2019-04-08

nmap

NMap，也就是Network Mapper，是Linux下的网络扫描和嗅探工具包。

nmap的主要功能

  1. 探测一组主机是否在线
  2. 扫描主机的端口，嗅探所提供的网络服务
  3. 推断主机所使用的操作系统

nmap 端口状态的判断

nmap的基础语法

  • namp -A [ip地址]

  • namp 192.168.1.1/24

  • nmap [ip地址1] [ip地址2]

  • nmap 192.168.1.1-100

  • nmap -iL 1.txt

  • nmap -sL 192.168.1.1/24

  • nmap 192.168.1.1/24 -exclude 192.168.1.1

  • nmap 192.168.1.1/24 -exclude 1.txt

  • nmap -p80,21,23 192.168.1.1

  • nmap -p 80-160 192.168.0.101

  • nmap 192.168.0.101,102,103

以上为nmap的最常用的基础命令

  • nmap -sP 192.168.43.8/24

  • nmap -P0 192.168.1.1

  • nmap -sS 192.168.1.1

  • nmap -sT 192.168.1.1

  • nmap -sU 192.168.1.1

  • nmap -sF 192.168.1.1

  • nmap -sV 192.168.1.1

  • nmap -sL 192.168.1.7 192.168.1.1

  • nmap -A 192.168.1.1

  • nmap -O 192.168.1.1

  • nmap -sA 192.168.1.1

  • nmap -PN 192.168.1.1

  • nmap -F 192.168.1.1

  • nmap -r 192.168.1.1

  • nmap –iflist

  • nmap -sN 192.168.1.1
